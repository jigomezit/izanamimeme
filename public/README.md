# Imágenes de Candidatos

Coloca aquí las imágenes de los candidatos con los siguientes nombres:

- `jimmy.jpg` - Imagen de Jimmy
- `alex.jpg` - Imagen de Alex  
- `mariano.jpg` - Imagen de Mariano

Las imágenes se mostrarán en formato circular en la página de votación.

Si no colocas las imágenes, se generarán automáticamente avatares con las iniciales de cada candidato.
