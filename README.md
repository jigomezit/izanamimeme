# Izanami Elige MOD 2025

Web estática con contador regresivo para la votación Izanami Elige MOD 2025.

## Tecnologías

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- shadcn/ui
- Aceternity UI - Background Beams With Collision
- canvas-confetti

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador.

## Build

```bash
npm run build
```

Esto generará una carpeta `out` con los archivos estáticos listos para desplegar.
